# Hospital Management Dashboard (Power BI)

## Overview

An advanced healthcare analytics dashboard developed using Power BI to monitor hospital operations, patient demographics, doctor performance, billing trends, treatment analytics, and executive KPIs.

## Tools Used

* Power BI Desktop
* DAX
* Power Query
* Data Modeling
* Excel/CSV

## Features

* Executive Overview Dashboard
* Patient Analytics
* Financial Analytics
* Doctor Analytics
* Executive Insights Dashboard
* Drill-through Reports
* What-if Analysis
* Dynamic KPI Measures
* Interactive Slicers

## Dataset

The project uses the following datasets:

* Patients
* Doctors
* Appointments
* Treatments
* Billing

## Key KPIs

* Total Revenue
* Revenue per Patient
* Revenue per Doctor
* Payment Success Rate
* Total Appointments
* Total Patients
* Total Doctors

## Advanced Features

* Date Dimension Table
* Time Intelligence DAX
* Drill-through Navigation
* Dynamic Analytics
* Forecasting & Scenario Analysis
